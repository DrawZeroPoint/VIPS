void VIPS_Model::add_meta_info_for_components ( int  N)
protected

The VIPS-model stores various meta-information for each component, e.g. KL bounds, the target distributions for the KL constraint and some debug data like the history of achieved rewards.
This function will enlarge the matrices/vector that store this meta-information in order to account for newly added components.

Parameters
N- the number of new components that have been added
